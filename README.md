## About the Project

This Python script automatically moves all `.jpg` files from a source folder to a destination folder.

## How It Works

* Moves them to another folder
* Uses `os` and `shutil` modules

## How to Run

1. Create a folder named `source_images` and add `.jpg` files.
2. Run the script:

   ```
   python move_jpg.py
   ```
